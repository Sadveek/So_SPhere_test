# So_SPhere_test

SocialSphere project workspace.

## Project Location
- Main app: `Social Media 2.0`

## Quick Start
```bash
cd "Social Media 2.0"
npm install
npm test
npm start
```

Open `http://localhost:4173`.

## What Is Implemented
- Email/password authentication
- Profile create/edit/view
- Post create/edit/delete
- Likes (idempotent)
- Comments
- Follow requests (send, accept, reject, cancel) and unfollow
- Feed from self + followed users
- Real-time updates with Firestore listeners
